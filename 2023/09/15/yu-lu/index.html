甘甘（数）语录

  • 眼巴神儿：约等于“眼力见儿”。
  • 紧接着：甘甘的口头禅。
  • 嗓子眼里塞鸡毛了：比喻说话前先清嗓子的同学。
  • 踏踏实实认认真真进入到学习状态：认认真真做事，踏踏实实做人。
  • 真正做到“明理懂事勤勉向上”：教室后面那字儿干啥用的？
  • 太岁爷头上动土：表达自己不受委屈，要大干一场。
  • 我是当班主任的，眼睛里不容沙子：开学第一天，下马威。
  • 我上届的一个学生：一大堆激励人的事迹。
  • 值日班长开始讲评：一大批同学奔赴刑场。
  • 入室即静、入座即学、入学即专：抄 100100100 遍给我。
  • 零抬头：自习课的要求。
  • 娘的 / 妈妈的 / 妈的：甘甘以此表示情绪激动，基本每天都要说。
  • 给我站着：同老邓“站好咯”。
  • 还没进门呢就先叫妈了：某些同学的“文明用语”。
  • 得有点儿精气神儿，干啥得像回事儿：薛定谔的猫不知死活，也有可能半死不活。
  • 一定要有自己的计划，心中有数：“数”借指数学。
  • 一做二订三改四签：作业的四个步骤。
  • 自奋蹄儿：老牛亦解韶光贵，不待扬鞭自奋蹄。
  • 不赖呆：自以为是。
  • 把每一道都弄懂弄透 / 打破沙锅问到底：指认真对待作业、对待学习。
  • 不蒸馒头争口气：字面意思。
  • 自律，每天进步一点点：1.01365≈37.781.01^{365}\approx37.781.01365≈37.78；1365=11^{365}=11365=1；0.99365≈0.030.99^{365}\approx0.030.99365≈0.03。
  • 贱白白 / 老先生：某孙姓同学的外号便是拜甘甘所赐。
  • 话多屁稠：字面意思。
  • 从已知到未知的过程：要体现出代入的过程。
  • 同桌之间交流一下（念叨念叨 / 叨叨叨叨）：互相交流加深印象。
  • 一二三 / 坐姿 / 来看黑板：该回来听课了。
  • 这个课间长我把这点讲完 / 多讲点：课间惨遭占用。
  • 谁再大声喊叫，值日班长给我重扣：课间有点太躁了啊！
  • 叨叨叨 / 啊啊啊 / 哈哈哈 / 嘿嘿嘿：甘甘的顶级模仿。
  • 你把炮弹都 duang~duang~duang~ 打出去：还普查呢。
  • 谁还在那嘿儿嘿儿直乐呢：讽刺笑点低的同学。
  • 我得咕噜到五班上课去 / 咕噜到后面站着去：约等于“滚”（含贬义）。
  • 你配吗 / 咱们副班长，狗屁不是吧：甘怒不敢言。
  • 大腿上桌子，你想上天啊：某同学跑完一千米之后把腿翘到桌子上了。
  • 你是跑了马拉松了：同上。
  • 你家只有一间屋了吗：讽刺。
  • 这不废话吗：这就是废话。
  • 高标准严要求：目标应该定在哪里呢？
  • 给我重扣 / 给他双倍扣分：甘甘生气了。
  • 这是错误的：答案不正确。
  • 在我这一亩三分地上，我说话还算数吗：算数算数，当然算数。
  • 现在咱教室也大了：后排的同学可能看不见，那我就更应该认真学习。
  • 好好练字儿 / 认真对待：对潘、刘宥言都说过，要认真对待板书。
  • 一放二贴三推四画：画平行线的步骤。
  • 挤肉饼 / 大肉包子：形容课间凑在一起的同学（含贬义）。
  • 新学期的要求是自律 / 还是那两个字自律：自律就是严于律己，不断自我反省，做到自觉遵守规则。
  • 定斩不饶：严惩不贷。
  • 敲山震虎 / 以儆效尤 / 杀鸡给猴看：但是猴看了吗？
  • 当牛做马：比喻被罚值日的同学。
  • 砖缝里的灰都给我舔掉：当牛做马。
  • 身边的榜样：也就是所谓的“大将”。
  • 打道回府：滚回家去。
  • 背着萝卜找擦床 / 找嚓嚓：甘甘认为“嚓”读作 cēi。
  • 缺德冒烟儿：某周姓同学被批评：不厚道。
  • 起哄架秧子：比喻起哄的同学。
  • 吊儿郎当 / 自以为是：比喻“成天自我感觉良好”的同学。
  • 我真是瞎了眼了：原来两任数学课代表都出现了不好的现象。
  • 死东西：别误会，指可以死记硬背的知识。
  • 咱班同学的那个笑声很诡异了：指直白地吃瓜。
  • 让班级多一点正能量 / 惜时：指学习。
  • 是骡子是马拉出来遛遛：借指“用成绩说话”。
  • 苍蝇不叮没缝的蛋：要从自身找原因。
  • 千叮咛万嘱咐：唠叨了几千遍，还是不听话。
  • 一提二套三分组，二次三项看十字：因式分解的方法（提公因式、套公式、分组分解、十字相乘）。
  • 有理有据 / 前有车后有辙：写数学时的注意事项。
  • 洗心革面重新做人：对某孙姓同学的衷心劝告。
  • 凭啥：语气强烈，此时的甘甘很生气。
  • 鸡蛋里头挑骨头：较真一把。
  • 这个班还承得下你吗：逐客令。
  • 刀子嘴豆腐心：形容历史老师。
  • 我打断你的腿 / 打折你的腿：再给我晃。
  • 一定一定要认真听讲：这是真理。
  • 比较简单：数学课时、练习、考试在甘甘眼里相对简单一些。
  • 你这样儿上不了高中：承接上文。
  • 我不同意（谁谁谁）去（干啥）：纪律是大事儿。
  • 比一比看一看：谁是踏踏实实认认真真学习的那个。
  • 人模狗样 / 说人话不干人事儿：直接的评价性语言。
  • 学会感恩：字面意思。
  • 她比我更年轻、更有活力、更有方法：形容新的班主任。
  • 虽然我不当班主任了：照样收拾你。
  • 把胳膊抻直了：原因是某李姓同学手势舞十分敷衍。
  • 有气无力 / 半死不拉活：文岐课间操动作不到位，被一顿臭骂。
  • 今天我把这小子臭骂了一顿：文岐。
  • 风雨同舟，荣辱与共：换班主任前的最后一句名言。

  • 冤有头债有主：谁干的我收拾谁。

  • 回望检查：确认无误后再接着做。
  • 五马张飞：马马虎虎。
  • 给我站着：字面意思。
  • 刷存在感：形容某韩姓同学。

  • 作业就写成这样啊：不行以后别写了。
  • 居里夫妇的五个精神，给我复述一遍：热情、专注、高效、分工合作、无怨无悔。

  • 我以前的班跑操，从来都是排头兵：今天看你们跑操，真是损我面子。
  • 跑到吉象石那喊口号，领导都捂着耳朵：张老师说过遇到巨大声响张嘴或捂耳。
  • 咱不说吓得领导屁滚尿流：领导直呼“666”。
  • 咱不能让别的班说十四班跑得屁滚尿流：甘甘的训斥。
  • 第二排李锦轩，面带笑容跑操：后边的我看不着，但是这是跑操的样子吗！
  • 目光炯炯：仿佛都要把前面跑道吃了一样。
  • 口号掷地有声：人家都喊“一二三四”，你们喊“一~二~三~四~~”。
  • 班委得起到带头作用：你们班委都干嘛去了，得做个表率啊。

  • 框外：不合理。
  • 纠错本作业就写成这样：咱们月考成绩见。
  • 我就佩服这样的人：指经验丰富，即使身体不好也从不请假坚持上课的老邓。
  • 反过来：反，读一声。
  • 我就在想啊：上班的路上我在想今天要干什么，进了班我在想先干什么后干什么。
  • 你们班久而久之，班风必定不正：不允许捕风捉影，一看见什么就吱儿喳乱叫。

老邓（语）语录

世有老邓，然后有小邓。小邓常有，而老邓不常有。故虽有小邓，祗辱于甘金梅之手，骈死于任丽娟之间，不以小邓称也。
——某人《邓说》
（老邓：哪有那么老。）

  • 每当有同学喊我老邓时，我就会想起年轻时同事叫我小邓：这就是“老邓”的由来。
  • 这还行啊 / 扣分：老邓生气了。
  • 还有天理吗：学生一致要求减少语文作业。
  • 不重视语文（同样也是某课代表名言）：潘？
  • 再这样我就狠狠地加作业 / 再吵吵就加作业：然而一次也没加过。
  • 每当有同学喊我老邓，我就会想起年轻的时候同事们叫我小邓：因此丁文岐撰文《邓说》。
  • 你看看：用于强调后面一句话的重要性。
  • 你看看这整的，这是啥啊：表示强烈不满。
  • 摇头摆尾：天宁。
  • 我们邓家人 / 我们小邓：小邓。老邓生气时会直呼其名。
  • 后面有哪个美女同学迷住你了：小邓一直往后看。
  • 小丁 / 小潘 / 宥言 / 苏文 / 禾禾 / 栋儿（国家栋梁）/ …：对班里同学的爱称。
  • 丁文丁：嘴瓢地读“丁文岐”。
  • 合起书 / 看黑板 / 抽查：要检查背诵的节奏。
  • 就知道要减语文作业：别的哪科多都与我大语文无关。
  • 说了多少遍两本两书，又去翻书包了：课前准备干啥了？
  • 这擦的叫黑板啊：比喻不负责任的值日生（以刘宥言为首）。
  • 以手为笔：谁还在那愣神儿。
  • 谁以后还敢在语文课上写其他科作业：作死（比如潘）。
  • 落到笔头上：说的挺好，考试为啥没分呢？
  • 欠收拾 / 欠打：表现不好。
  • 诶，哪还有声音啊（这是）：约等于“保持安静”。
  • 心浮气躁 / 踏踏实实：着急嘛慌 / 稳稳当当。
  • 人家拿第一不是没道理的：锦轩被示众。
  • 都看看人家（谁谁谁）的（啥啥啥）：表扬谁谁谁的啥啥啥。
  • 优加真棒 / 乱而差（练字去）：冰火两重天。
  • 怎么还不自信了：语文需要沉淀（潘的描述）。

  • 答题卡先生——高嘉泽：指文岐。
  • 液体压强先问题——丁文岐：文岐的口误。
  • 甲古（骨）文、乙古文……苏文——丁文岐：文岐发现的新大陆。
  • 再这样我让你们吃雪去——甘甘：原来甘甘也是 homo（喜）。
  • 四人帮——甘甘：郝明辉、冯屿硕、高嘉泽、侯慕宸构成的小集体。
  • 田忌 / 上等马 / 中等马 / 下等马——郝明辉：指“四人帮”中的成员。
  • 黑老（牢）大——甘甘：指郝明辉。
